这是一个AI直接操作电脑系统的项目，通过抽象各个层级，并且提供对外接口，可以使开发者快速开发各个插件，实现自己想要的功能。   
整个项目运行在Dokcer中，不用担心AI把电脑搞坏了，如果出问题了，直接重新构建即可。  
具体详情请[查看文档](./README-cn.md) 。  

主要功能：
- 对外开发WebUI、SSH、IM等入口与AI交互。  
- 通过大模型ApiKey配置，可以接入任何大模型。
- 运行在Docker中，不怕AI搞坏电脑，并且有价值的内容可以存储在宿主机中。
- 使用多AI协助，提高工作效率。
- 拥有记忆压缩功能，删除无效记忆，增加上下文。
- 分层设计，每层都有对外接口，方便后续开发以及扩展。
- MIT协议，所有人都可以免费快速接入，而不用担心权限问题。
